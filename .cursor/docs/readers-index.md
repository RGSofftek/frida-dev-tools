## Reader Categories

### Office Automation
- **[Excel Reader](excel-reader.md)** - Spreadsheet manipulation, formulas, filtering, tables
- **[Word Reader](word-reader.md)** - Document creation, templates, tables, images
- **[Powerpoint Reader](powerpoint-reader.md)** - Presentation automation

### Enterprise Applications
- **[SAP Reader](sap-reader.md)** - SAP GUI automation via SAP Scripting
- **[DB Reader](db-reader.md)** - Database connections and queries (SQL Server, MySQL, Oracle, SAP HANA, DB2)

### Communication
- **[Outlook Reader](outlook-reader.md)** - Email automation without Outlook installation

### Web & UI Automation
- **[Browpet Reader](browpet-reader.md)** - Web browser automation
- **[Titanium Reader](titanium-reader.md)** - Desktop UI automation, keyboard, mouse, images

### Core & System
- **[Mix Reader](mix-reader.md)** - Core RPA logic: variables, loops, conditionals, error handling
- **[File Reader](file-reader.md)** - File system operations: copy, move, read, write, zip
